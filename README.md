# 數值記錄計算中心

個人使用的數值計算與記錄小工具，單一 HTML 頁面，資料保存在瀏覽器 `localStorage`，無需後端或登入。

## 如何運行

### 方式一：直接開啟（推薦）

用瀏覽器開啟專案根目錄的 `index.html` 即可使用。

### 方式二：本地靜態伺服器（可選）

若需避免部分瀏覽器對 `file://` 的限制，可在專案目錄執行：

```bash
npx --yes serve .
```

或使用 Python：

```bash
python -m http.server 8080
```

然後在瀏覽器開啟 `http://localhost:8080`（埠號依實際輸出為準）。

## 功能說明

- **即時計算**：修改「目標值」或「當前值」後，六項計算結果立即更新（輸入為空時視為 0）。
- **計算規則**：
  - 原始值 = `(目標值 − 當前值) ÷ 10`（整數除法，向零取整）
  - 0.5 / 0.34 / 1.5 倍：對乘積向零截斷取整（與 Kotlin `toInt()` 一致）
  - 2 倍、3 倍：整數乘法
- **輸入校驗**：僅接受整數；非法輸入會提示並將結果歸零。
- **保存記錄**：將當前輸入與計算結果寫入本地；保存時目標值、當前值不可為空。
- **歷史摘要**：顯示上次成功保存的完整記錄；無保存過時顯示「暫無歷史記錄」。
- **刷新還原**：重新載入頁面後自動從 `localStorage` 還原欄位與歷史區。

## localStorage 鍵名

| 鍵名 | 說明 |
|------|------|
| `target_value` | 目標值 |
| `current_value` | 當前值 |
| `calculated_value` | 原始計算值 |
| `calc_0_5x` | 0.5 倍結果 |
| `calc_0_34x` | 0.34 倍結果 |
| `calc_1_5x` | 1.5 倍結果 |
| `calc_2x` | 2 倍結果 |
| `calc_3x` | 3 倍結果 |
| `record_saved` | 是否曾有成功保存（`1` 表示有歷史記錄） |

## 技術選型

採用 **純 HTML + CSS + JavaScript**，無建置步驟、無框架依賴，可直接雙擊開啟，適合 GitHub Pages 等靜態託管。
